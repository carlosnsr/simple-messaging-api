# Simple Messaging API

A simple messaging API done for my [Guild Education] technical test.

[Guild Education]: https://www.guildeducation.com/

## Configuration

### Installation

```
bundle
```

### Database creation

```
rails db:migrate
```

## To run

```
rails server
```

It is now running on `localhost:3000`

## Test suite

To run the included test suite:

```
rspec
```
